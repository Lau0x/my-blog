# v1.3.0 · 说说 + 全局搜索 + 新手手册

> 发布日期：2026-04-19 · [GitHub Release 页面](https://github.com/Lau0x/my-blog/releases/tag/v1.3.0)

第三个版本。聚焦**新内容形态 + 信息组织 + 新手友好**。这一版让非技术用户也能轻松维护博客：内容类型多了一种（说说），前后台都加了搜索，并有一份写给朋友的零前置手册。

## ✨ 新功能

### 💬 说说 / Notes（新内容形态，类 Twitter 轻量记录）
- 新 Collection Type **说说（Note）**：content (markdown) + media (多图 + 视频) + tags + publishedDate
- 前台新页面 **`/notes`**：时间轴式瀑布流
  - 左侧垂直轴 + 节点圆点（仿 GitHub contribution graph 布局）
  - 图/视频自动网格排布（1/2/3/4 张，twitter 风）
  - Markdown 渲染（代码高亮、外链自动新窗口、图片 lightbox 全部继承）
  - **右侧 sticky 时间线导航**（≥1200px）按年月快速跳转
- 视频上传限制 **800MB**（中间件 + upload plugin 两层一致）

### 🔍 全局搜索
- 新页面 **`/search`**：输入框 + URL 同步（`?q=xxx` 可分享）+ 300ms debounce
- 同时搜**文章**（title / excerpt / content）和**说说**（content）
- 关键词自动 `<mark>` 高亮，预览片段去除 markdown 噪音
- Header 右上角 🔍 图标入口

### 🧙 管理员权限兼容（非破坏性）
- 新 content-type 对 Public role 默认无权限 — `scripts/upgrade.sh` 末尾新加提醒，指引 admin 手动开 find/findOne（v1.2 发现的漏洞已补）

## 📚 文档

- 新增 [`docs/newbie-guide.md`](docs/newbie-guide.md) — **给非程序员朋友的零前置 admin 手册**：
  - 关键概念：前台/后台、Collection vs Single、Save vs Publish
  - 发第一篇文章 · 发说说 · 改站点名 / 导航 / footer
  - 建 CustomPage（markdown 和 raw HTML 双模式）
  - 4 个常见坑（缓存、权限、草稿、上传大小）
  - admin 中文化 + 字段 label 自定义
- README.md 索引新增

## ⚙️ 工程改进

- `backend/config/middlewares.ts` — strapi::body `formLimit`/`jsonLimit`/`textLimit` 全部 800MB，`formidable.maxFileSize` 同步
- `backend/config/plugins.ts` — upload plugin `sizeLimit` 800MB

## 📋 升级指南（有权限操作，非破坏性）

```bash
# Step 1. 拉新版镜像
ssh <你的博客服务器>
cd /path/to/blog
vim .env                    # IMAGE_TAG=v1.3.0（推荐 pin）
./scripts/upgrade.sh        # 脚本会自动提醒你做 Step 2
```

```
Step 2. admin 开权限（必做，否则前端说说/搜索看不到）

https://<你的域名>/admin → Settings → Users & Permissions → Roles → Public → 勾：
  - Note: find + findOne      （新增）
  # v1.2 时开过的 Article / Site-config / Custom-page 保持不变
```

```
Step 3. 可选：如果 NPM 反代生效，NPM 里上传大小限制要同步

NPM Proxy Host → /api 和 /admin 的 Custom Location → Advanced：
  client_max_body_size 800m;

否则 800MB 视频会在 NPM 层被拦截。
```

**数据层面**：Strapi 启动时自动 `ALTER TABLE` 创建 `notes` 表和关联表，不影响现有数据。
