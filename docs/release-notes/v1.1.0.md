## ✨ 新功能

- **前台视觉升级** — LXGW 文楷 / Inter / JetBrains Mono 字体，Markdown 样式重做
- **代码块增强** — 语言标签 + 一键复制按钮 + highlight.js 语法高亮
- **图片点击放大** — medium-zoom 实现的文章内图片 lightbox
- **首页卡片重设计** — 封面背景图 + 左白右透的渐变遮罩
- **图片策略双选** — 外链默认零配置（带 `referrerpolicy="no-referrer"` 绕防盗链）/ 一键本地化脚本 `scripts/migrate-images.mjs`（幂等 + 断点续跑 + cron 友好）
- **Tag 从 JSON 升级为 Repeatable Component** — admin 里点 `+ Add an entry` 加标签，非技术用户也能用
- **评论区脚手架（Giscus）** — 配 `giscusRepoId` / `giscusCategoryId` 即可启用，默认关闭

## 🐛 修复

- Markdown 代码块双转义（`&amp;amp;amp;lt;span` 渲染成字面量）
  - 根因：marked 插件在 `<script setup>` 里每个组件实例重复注册，堆叠 N 次转义
  - 方案：抽出 `frontend/utils/markdown.ts`，`marked.use()` 只在模块加载时跑一次
- 首页封面图不显示 — `useAdConfig.imgUrl()` 原只接受 object，不兼容 string 入参
- 第三方 CDN（mdnice 等）防盗链 403 — 所有 `<img>` 注入 `referrerpolicy="no-referrer"`

## ⚠️ 升级指南（有破坏性变更）

本版 `tags` 字段从 `json` 改为 Repeatable Component，**Strapi 5 不会自动迁移**，必须手动处理：

```bash
# ① 先备份（务必执行，破坏性操作不可逆）
./scripts/backup.sh

# ② 手动 DROP 老的 jsonb 列，让 Strapi 重启后按新 schema 建 articles_cmps 关联表
docker compose exec postgres psql -U blog -d blog \
  -c "ALTER TABLE articles DROP COLUMN tags;"

# ③ 拉新镜像 + 重启
./scripts/upgrade.sh
```

如果原来在 `tags` 字段里填过 JSON 数组（`["医路","思考"]` 这种），这些数据会丢失，需要在新 UI 重新录入。文章本体（title / content / cover）**不受影响**。
